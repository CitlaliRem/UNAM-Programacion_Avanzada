Descripción:

El programa utiliza el primer renglón como pivote para poder obtener una constante,
la cual multiplica al renglón pivote y el resultado del producto se le resta a los
valores de los renglones posteriores.

Después se hace algo similar pero ahora de abajo hacia arriba de la matriz.
