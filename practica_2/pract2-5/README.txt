Descripción: 

Este programa transpone una matriz de 4x4 utilizando solamente un apuntador. Haciendo aritmética con
el apuntardor se encuentran las posiciones del arreglo y se efectuan los conmutaciones requeridos.
La variable "cycle" sirve para que los for loops puedan visualizar la matriz en forma cuadrada.
