Descripción: 
Genera java.lang.ArithmeticException: / by zero. 
Se generá porque no se puede dividir por 0 Esta execption se generá con datos
primitivos int, en el caso de double la exception no se genera. 
Explicación de qué tipos de datos pueden generar una exception: ArithmeticException. 
Según la documentación de Oracle, se hace referencia, de forma específica, a la
division por cero con datos int.
