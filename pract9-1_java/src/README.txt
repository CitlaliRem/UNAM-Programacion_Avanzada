Descripción: 
Programa que recibe dos números complejos, y obtiene la multiplicación.
Este programa implementa unas clases que formatean los mensajes para la entrada
de datos. Según el número y la parte de éste que se requiere entrar el programa
reemplaza un 1 por un "first" y el 2 por un "second". Lo mismo sucede con el la
parte "real" y "imaginary".
La idea era obtener dos volores de retorno de una misma función.

Programación avanzada y métodos numéricos.
