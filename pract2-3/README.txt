Descripción: 
Se tiene un programa con las siguientes declaraciones:
int num;
int *p;

Suponiendo que la dirección de num es 7753 y la de p 8364, determinar los siguientes valores:
a) num
b) p
c) &num
d) &p
e) *p

Después de ejecutar de forma acumulativa, las siguientes sentencias:

A. num = 5; p=12

a)num = 5
b) p =12
c) &num = 7753
d) &p = 8364
e) *p = &num


B. num = p

a) num = 12
b) p =12
c) &num = 7753
d) &p = 8364
e) *p = &num


C. num = &p

a) num = 8364
b) p =12
c) &num = 7753
d) &p = 8364
e) *p = &num


D. p = &p

a) num = 8364
b) p = 8364
c) &num = 7753
d) &p = 8364
e) *p = &num

E. p = &num

a) num = 8364

d) &p = 8364
e) *p = &num
