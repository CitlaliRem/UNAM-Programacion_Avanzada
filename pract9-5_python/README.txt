Descripción: 
Este progrma permite la entrada dinámica de una matriz cuadrada de tamaño
arbitrario. El usuario ingresa la primera fila y confirma con "q" cuando termina
está misma. Después el programa calcula el tamaño y pide los valores que faltan.
Si el usuario ingresa un valor que no sea un float, el programa vuelve a pedir
el valor para este campo. En case de que el usario ingresa 3 veces mal, el
programa termina advertiendo que no se debe jugar con él :)
Al final evalua automáticamente la matriz si se trata de valores positivos,
negativos o nulos para cada campo y da un reporte detallado y un resumen.

Observaciones:
El programa está completamente impplementado de forma orientado a objetos y
utiliza conceptos de tratamientos de excepciones para el ingreso de valores controlado.

Programación avanzada y métodos numéricos.
