Descripción: 
Este programa implementa un algoritmo de Eliminación Gausseana para una matríz
de 3x3 y encuentra el valor de x_1.
Para asegurar que el resultado sea correcto aún cuando el pivote se hace cero,
corre un test antes que detecta ese caso. Una vez detectado un pivote en cero
cambia renglón en cuestión con el que sigue, solamente en el caso de que no sea
el último renglón.
